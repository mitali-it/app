  # 📌Features
  🎧 Play songs using audio player
  
  ▶️Play and pause songs
  
  ⏯️Skip to next track
  
  ⏭️Go back to previous track
  
  🔊Volume Control
  
 📂Multiple song list

 💫Fast and lightweight

 🎨 CSS
